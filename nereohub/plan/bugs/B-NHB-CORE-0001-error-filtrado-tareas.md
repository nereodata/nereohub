### Descripción
En el backlog del plan de ejecución, las tareas no se filtran correctamente cuando se aplican los filtros de proyecto u otros filtros disponibles.

### Pasos para reproducir
1. Abrir NereoHub.
2. Navegar al plan de ejecución.
3. Intentar filtrar por un proyecto específico.
4. Observar que la lista de tareas no cambia.

### Descripción
No se permite seleccionar o escribir una versión directamente en la etiqueta de la tarea. Esto es necesario cuando un proyecto aún no tiene versiones definidas o se desea asignar una versión nueva sobre la marcha.

### Pasos para reproducir
1. Intentar editar el campo de versión de una tarea en la vista de plan.
2. Observar que no hay opción para introducir una versión nueva si no existe previamente.
